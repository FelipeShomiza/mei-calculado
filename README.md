# MEI Calculado

Site brasileiro com calculadoras e guias informativos para Microempreendedor Individual (MEI).

O projeto oferece ferramentas simples para estimar limite de faturamento, limite proporcional, DAS mensal, projeção de faturamento e relatório mensal do MEI.

> O MEI Calculado é um site independente e não possui vínculo com o Governo Federal, Receita Federal, Portal do Empreendedor ou Sebrae. As informações são apenas informativas e devem ser conferidas em fontes oficiais.

## Site publicado

https://meicalculado.com.br

## Funcionalidades

- Calculadora de limite MEI proporcional
- Calculadora de faturamento MEI
- Calculadora DAS MEI
- Relatório mensal MEI
- Guias informativos sobre limite, DAS, nota fiscal e faturamento

## Stack

- Next.js com App Router
- TypeScript
- Tailwind CSS
- Export estático
- Cloudflare Pages
